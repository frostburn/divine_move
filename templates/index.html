Note that the rules of the games have been slightly modified to be more elegant from a game theoretic perspective. This can make things a little strange at times when repeating positions are involved.

Everything is still pretty much in beta stage right now. Here's a list of features planned if there's enough interest.

  • Registering an actual domain
  • User accounts
  • Rating positions
  • Commenting on positions
  • Favorites and collections
  • Elo type difficulty rating for positions and a corresponding skill rating for users
  • More fully editable categories like the most important 5 piece chess endgames and 5x4 go
  • Non-editable positions with a book answer for every possible user move. These include arbitrary chess problems with relatively low move limits, 5x5 go and carefully crafted go problems with immortal stones and limited playing area that make finding perfect solutions feasible.
  • Standard human friendly rulesets
